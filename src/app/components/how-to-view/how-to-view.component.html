Welcome to this section. Here I will laout the steps on how to implement a custom PrimeNg theme into your angular application.

First we need to download the Theme related SCSS files from PrimeNg. For that you can visit this link. Copy the 'theme-base' folder and put it under assets folder (may be make a new folder like 'primeng-theme' under assests folder and put this under nwly created folder). Now create another 'themes' folder where you can keep all your theme related variable, font and extension files.

Create 2 separate theme files, under 'themes' folder, for Light and Dark mode variables (_dark-variables.scss and _light-variables.scss). These files will hold the variables that we will use to customize the PrimeNg components.

Now under the 'themes' folder, create another folder called 'custom-themes' that will house your custom theme files.

You can refer this repo to refer the content of what the above 2 dark and light variable files will hold. You can also refer to any of the custom theme files and go through its content, customize the colors as per your need. One thing to point out, each of the custom theme files work upto line 17. The rest of the CSS variables are just reference colors.
